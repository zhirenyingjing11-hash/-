# Amazonアフィリエイト 自動投稿キット

商品（ASIN/検索ワード）と自分のメモを書くだけで、**note・Instagram・Threads の3媒体の文章をAIが自動生成**し、**Amazonアフィリエイトリンクを付与**して、**Threads と Instagram には自動投稿**するための Google Apps Script キットです。

> note は公式の投稿APIが存在しないため「完全自動投稿」はできません。本キットは note 用の記事を自動生成し、`noteドラフト` シートにコピペできる形で出力します。

---

## できること

| 媒体 | 文章生成 | アフィリンク付与 | 自動投稿 |
|---|---|---|---|
| **Threads** | ✅ AI生成（500字以内） | ✅ 末尾にリンク | ✅（既存トリガー） |
| **Instagram** | ✅ AI生成＋ハッシュタグ | ✅ プロフィール誘導＋リンク保持 | ✅ Graph API |
| **note** | ✅ AI生成（1500〜2500字の記事） | ✅ 本文末に埋め込み | ⬜ 手動コピペ（API無し） |

- **商品情報の自動取得**: Amazon PA-API v5 で商品名・価格・画像URL・アフィリンクを取得（AWS署名V4を内製、外部ライブラリ不要）。
- **PA-APIなしでも動く**: PA-APIのキーが未設定なら自動で「手入力モード」に切替。商品名はB列に入力、アフィリンクは ASIN/検索ワードから自動生成。審査が通ってキーを入れれば、何もせずフル機能に切り替わります。
- **AI生成**: Gemini（既定）または Claude に切替可能。
- 既存の `threads-auto-post.gs` / `threadsweekprep.gs`（Threads自動投稿・食レポ運用）を壊さず拡張します。

## ファイル構成（`gas/`）

| ファイル | 役割 |
|---|---|
| `appsscript.json` | マニフェスト（権限スコープ・タイムゾーン） |
| `00_config.gs` | 設定・シークレット管理・既存ファイルとの橋渡し |
| `10_amazon_paapi.gs` | Amazon PA-API v5（AWS署名V4で商品取得） |
| `20_ai_writer.gs` | Gemini / Claude で3媒体の文章を生成 |
| `30_instagram.gs` | Instagram Graph API 自動投稿 |
| `40_orchestrator.gs` | 司令塔（取得→生成→各シートへ書き出し） |
| `50_menu.gs` | スプレッドシートの統合メニュー（`onOpen`） |
| `threadsweekprep.gs` | 既存の食レポ運用キット（`onOpen` は 50 に統合済み） |

> `threads-auto-post.gs`（Threadsへの実投稿・トークン管理）はあなたの既存プロジェクトのものをそのまま使ってください。本リポジトリには含めていません。

## セットアップ手順

### 1. ファイルを Apps Script に追加
お使いの Google スプレッドシートの拡張機能 → Apps Script で、`gas/` 内の各 `.gs` を**同名で新規ファイル**として貼り付けます。`appsscript.json` の `oauthScopes` も反映してください（clasp利用ならそのまま push）。

> ⚠ `onOpen` は1プロジェクトに1つだけ。既存 `threadsweekprep.gs` の `onOpen` は削除済みで、`50_menu.gs` に統合しています。

### 2. APIキーを「スクリプトプロパティ」に登録
Apps Scriptエディタ →（左）⚙ プロジェクトの設定 → スクリプトプロパティ で以下を登録（または `setSecret('キー','値')` を実行）:

| キー | 内容 | 必須 |
|---|---|---|
| `AMAZON_ACCESS_KEY` | PA-API アクセスキー | ✅ |
| `AMAZON_SECRET_KEY` | PA-API シークレットキー | ✅ |
| `AMAZON_PARTNER_TAG` | アソシエイトID（例 `yourid-22`） | ✅ |
| `AI_PROVIDER` | `gemini`（既定）または `claude` | 任意 |
| `GEMINI_API_KEY` | Gemini を使う場合 | ◯ |
| `CLAUDE_API_KEY` | Claude を使う場合 | ◯ |
| `IG_ACCESS_TOKEN` | Instagram 長期アクセストークン | IG投稿時 |
| `IG_BUSINESS_ID` | Instagram ビジネスアカウントID | IG投稿時 |

登録後、メニュー **アフィリ自動投稿 → 設定チェック** で過不足を確認できます。

### 3. シートを準備
メニュー **アフィリ自動投稿 → ① シートを準備** を実行すると、`商品マスタ` / `IGポスト` / `noteドラフト`（および無ければ `投稿`）が作られます。

### 4. Instagram 自動投稿トリガー（任意）
メニュー **アフィリ自動投稿 → Instagram自動投稿トリガー設置** で15分おきに `IGポスト` を巡回し、予約時刻が来た行を投稿します。

## 使い方（日々の運用）

1. `商品マスタ` シートに記入
   - **A列**: ASIN（例 `B0XXXXXXXX`）または検索ワード
   - **F列**: あなたのメモ（実体験・推しポイント。AIが自然に反映します）
   - 手入力モードのときは **B列に商品名**（任意でC価格・D画像URL）も入れると文章の精度が上がります
2. メニュー **アフィリ自動投稿 → ② 文章を一括生成** を実行
   - PA-APIで B〜E列（商品名・価格・画像・アフィリンク）が自動入力
   - AIが3媒体の文章を生成し、各シートへ振り分け
     - Threads → `投稿` シート（既存の自動投稿トリガーが時間になったら投稿）
     - Instagram → `IGポスト` シート（IGトリガーが投稿）
     - note → `noteドラフト` シート（本文をコピーして note に貼り付け）
3. 投稿時刻は翌日以降の空き枠（`DAILY_TIMES`）に自動割り当て。変更したい場合はシートの日時セルを直接編集。

> 再生成したい時は `商品マスタ` の **G列（状態）** を空にして再実行してください。

## 媒体ごとのリンクの扱い（規約・作法に配慮）

- **Threads**: 本文中にURLを入れず、末尾にアフィリンクを配置（最初の返信に貼る運用にも流用しやすい形）。
- **Instagram**: キャプション内のURLはリンクになりません。「プロフィールのリンクから」と誘導し、アフィリンクは `IGポスト` のE列に保持（リンクインバイオ等に設定）。
- **note**: 本文末にアフィリンクを埋め込み、「アフィリエイトリンクを含みます」の表記を自動付与。

---

## 注意・制限

- **note は手動投稿**（公式APIが無いため）。生成した本文を `noteドラフト` からコピペしてください。
- **PA-API** は審査・直近の売上実績が必要です。利用できない間は `threadsweekprep.gs` の `AFFILIATE()` 関数や `buildAffiliateUrl_()` で簡易リンクも作れます。
- **Instagram自動投稿** はプロアカウント＋Facebookページ連携＋公開画像URLが前提です（Amazonの商品画像URLが使えます）。
- 薬機法・景表法に触れる誇大表現を避けるよう、AIプロンプトで断定的な効能表現を抑制しています。最終確認は必ず人の目で。
- 各プラットフォームの利用規約・自動化ポリシーは変わる可能性があります。運用前にご確認ください。

## カスタマイズの勘所

- 投稿時間帯: `threadsweekprep.gs` の `DAILY_TIMES`
- AIモデル: `00_config.gs` の `GEMINI_MODEL` / `CLAUDE_MODEL`
- 文章のトーン・構成: `20_ai_writer.gs` の `buildWriterPrompt_()`
- シート名: `00_config.gs` の `SHEET_*`

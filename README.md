# 電力設計 13大計算ツール

電気設備設計の**概算検討・学習用**の計算ツール集です。1ファイルの HTML（`index.html`）だけで動作し、スマホ・PC どちらのブラウザでも使えます（レスポンシブ対応・PWA/オフライン対応）。

> ⚠️ **免責**: 本ツールは概算・学習用です。実設計・申請は最新の内線規程(JEAC8001)・電技解釈・JIS・メーカーカタログ・所轄/電力会社協議で必ず確認してください。

## 収録ツール

| # | ツール | 内容 | 主な根拠 |
|---|--------|------|----------|
| ① | ケーブル・電線選定 | CV/VV/IV/DV/OW/OE の6種別対応。電圧はプルダウン選択（気中/管内/地中直埋/管路） | JCS 0168-2 / 内線規程1310・1340 |
| ② | ブレーカ選定 | 連続負荷×1.25 → 標準定格を選定（Iwは種別×布設条件から自動計算） | JIS C8201相当 / 内線規程 |
| ③ | ケーブルラック幅 | ケーブル種類・サイズをプルダウン選択 → 必要ラック幅 | 設計慣行（占有率） |
| ④ | 電線管・樹脂管 | IV・CV・VVR をプルダウン選択 → 占積率 → 管サイズ | 内線規程3110 |
| ⑤ | 照度計算（光束法） | 器具プリセット＋照明率/保守率の目安自動入力＋簡易配置図 | JIS Z9110 / 光束法 |
| ⑥ | ちょう架（吊架）線 | 材質（AC/ST）・ケーブル質量・たるみ目安から必要太さを自動選定 | 電技解釈 / 内線規程 |
| ⑦ | コンクリートトラフ | 条数・外径 → トラフ呼び名（占積率上限は任意） | JIS A5372 |
| ⑧ | 変圧器容量選定 | 設備容量×需要率 → 標準容量 | JIS C4304/C4306 |
| ⑨ | 力率改善コンデンサ | 目標力率までの必要kvar → 標準容量 | JIS C4902 |
| ⑩ | 短絡電流（%Z法） | 変圧器%Zから短絡電流を概算・Icu判定 | %インピーダンス法 |
| ⑪ | B種接地抵抗（簡易） | 1線地絡電流 → 接地抵抗の上限値（Igのこう長概算つき） | 電技解釈17条 |
| ⑫ | 接地工事（A〜D種） | 種別 → 抵抗上限＋接地線太さ＋大地抵抗率から接地棒の概算本数・平面図 | 電技解釈17・29条 / 内線規程1350 |
| ⑬ | 盤基礎（耐震簡易） | 形式（ベタ/ボイド/ゲタ）・転倒チェック・アンカー選定＋平面図/側面図 | 建築設備耐震設計 簡易法 |

ほかに **用語集（47語・検索つき）** と、各入力欄の「?」からのインライン用語解説つき。③⑦は CV3心/CVT/CV2心/CV単心/VVR、④は IV をプルダウン選択で外径自動入力（参照表も内蔵）。

## 使い方

- **PC**: `index.html` をブラウザで開くだけで動きます（幅900px以上では入力と結果が左右2カラム表示になります）。
- **スマホ**: GitHub Pages などで公開した URL を開き、「ホーム画面に追加」でアプリのように全画面で使えます（オフライン対応）。
- 入力値は端末ごとに自動保存されます。ヘッダーの「↺」ボタンで初期値に戻せます。

### GitHub Pages で公開する

リポジトリの **Settings → Pages → Branch: main / (root)** を選ぶだけで `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開できます。

## ファイル構成

```
index.html    … アプリ本体（データ・計算ロジック・UI すべて入り）
manifest.json … PWA マニフェスト（ホーム画面追加用）
sw.js         … Service Worker（オフラインキャッシュ）
icons/        … アプリアイコン（SVG + PNG 180/192/512）
```

## カスタマイズのポイント（`index.html` 内）

- `DATA` … 許容電流表・管内径・標準定格などの**参照データベース**（出典つき）。値の追加・差し替えはここ。
- `GLOSSARY` … 用語集。`["用語","対応ツール番号","解説"]` を1行足すだけで検索・ツールチップに反映されます。
- `TOOLS` / `calcXxx()` / `resultXxx()` … ツールを増やす場合はこの3点セットに加え、`inputCard`・`bindInputs` の対応表・`recalc` に1行ずつ登録します。
